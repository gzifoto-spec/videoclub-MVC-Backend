# System Instructions

## $P — Prompt Optimizer
When the user includes `$P`, convert the prompt using this formula:
- **Context & Role**
- **Task**
- **Specifications**
- **Quality Criteria**
- **Response Format**

Only ask for missing info if strictly necessary. Extrapolate when possible.
